# Rote Agenda

Rote Agenda ist ein mobile-first MVP für Capture-first Aufgaben- und Projektorganisation. Rohnotizen werden lokal von einer Mock-KI in strukturierte Aufgaben, Projektzuordnungen, Deadlines, Prioritäten und prüfbare Vorschläge übersetzt.

## Kernflow

- Notiz schnell erfassen
- Mock-KI verarbeitet und klassifiziert die Rohnotiz
- Vorschlag prüfen, bearbeiten, übernehmen oder ignorieren
- Aufgabe erscheint im passenden Projekt und auf dem Heute-Dashboard
- Aufgaben abhaken und manuell bearbeiten

## Tech Stack

- Next.js App Router
- React
- TypeScript
- Tailwind CSS
- lucide-react
- lokale Mock-Daten und localStorage

## Entwicklung

```bash
npm run dev
```

Danach http://localhost:3000 öffnen.
